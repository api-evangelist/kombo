# Kombo (kombo)
Kombo is a unified API for HR and ATS integrations, enabling B2B SaaS companies to connect with HRIS, payroll, recruiting, and learning systems through a single integration.

## APIs

### Kombo Unified API
Connects B2B SaaS products with HR, payroll, applicant tracking, and learning management systems through a single integration point.

- **Human URL:** https://www.kombo.dev
- **Base URL:** https://api.kombo.dev
- **OpenAPI:** https://api.kombo.dev/openapi.json

### Kombo Unified HRIS API
Employee data, absence management, time-off, documents, and provisioning across HR systems.

### Kombo Unified ATS API
Job postings, candidates, applications, and recruitment pipelines across ATS platforms.

### Kombo Unified ATS-Assessment API
Assessment writing, notifications, and result submission across ATS platforms.

### Kombo Unified LMS API
User management, courses, and learning progress tracking across LMS platforms.
